Entregable 1 Jairo Bargas

El siguiente proyecto buscará implementar la idea de crear un sitio para solicitar turnos para el área de servicios en un concesionario de maquinaria agrícola John Deere. Para dicha plataforma los usuarios deberán ingresar con el nombre de usuario y contraseña que se les brinda al adquirir una máquina y luego seguir los pasos para la agenda del mismo.
